# ACM-Techstorm

ACM-Techstorm is a modern, responsive event website for the ACM Techstorm event, featuring a clean UI, interactive schedule, event details, and registration system. Built with HTML, CSS (including TailwindCSS), and JavaScript, it provides a seamless experience for both desktop and mobile users.

## 🌟 Features
- Modern, responsive UI with TailwindCSS
- Animated event cards and schedule timeline
- Event registration popup and confirmation
- Mobile-friendly navigation and layout
- Custom event and schedule scripts

- CSS3 (TailwindCSS, custom CSS)
